DUPLICATI

Duplicati es una aplicación de código abierto diseñada para realizar copias de seguridad de datos en sistemas informáticos. Sus características principales son: capacidad de programar y automatizar copias de seguridad, cifrar datos para la privacidad y la seguridad, se pueden almacenar las copias en varios destinos (nube, unidad local, servidor externo, etc.)

Ventajas:
  • Código abierto y gratuito
  • Cifrado de datos
  • Flexibilidad en el almacenamiento
  • Programación y automatización
  • Historial de versiones
  • Integración con servicios de sincronización en la nube
Desventajas:
  • Configuración inicial compleja (sobre todo elegir qué cifrado usar)
  • Recursos del sistema (sobre todo durante el cifrado y compresión de datos)
  • Posible inestabilidad (salvada por el constante desarrollo al ser de código abierto, por lo que hay que tenerla actualizada)
Requisitos del Sistema:

Duplicati debe instalarse en un dispositivo con un sistema operativo compatible. Actualmente, estos sistemas operativos son compatibles:

  • Windows Vista y superior (versiones de 32 y 64 bits)
  • Windows Server 2008 y versiones posteriores (versiones de 32 y 64 bits)
  • Linux
  • Apple Mac OSX
Duplicati necesita unos 40 MB de espacio libre en el disco duro para su instalación. Sin embargo, se requiere espacio adicional para la ejecución.
